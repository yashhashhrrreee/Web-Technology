My Work

Project 1: Online Bookstore Application

This project involved developing an online bookstore application where users could browse through a wide range of books, view book details, and purchase books using an integrated payment gateway. The application was built using Java and Spring Framework, and the front-end was designed using HTML, CSS, and JavaScript. The project also involved integrating the application with Amazon Web Services (AWS) for hosting and managing the application.

Some of the key features of the application included a search functionality to find books by title or author, a shopping cart to add and remove books, and a user account system for managing orders and tracking shipments. The application also included an admin panel for managing book listings, inventory, and order processing.

Project 2: Healthcare Management System

This project involved developing a healthcare management system for a hospital. The system was designed to streamline various hospital processes such as patient registration, appointment scheduling, and medical record keeping. The system was built using Python and Django Framework, and the front-end was developed using HTML, CSS, and JavaScript.

Some of the key features of the system included a patient portal for managing appointments and medical records, an administrative dashboard for managing patient information, and a pharmacy management system for tracking medication inventory and dispensing medication to patients. The system was also integrated with a payment gateway for processing hospital bills and insurance claims

Project 3: Mobile Application for Restaurant Ordering

This project involved developing a mobile application for a restaurant chain that allowed customers to place food orders from their mobile devices. The application was built using React Native for both Android and iOS platforms, and the back-end was developed using Node.js and Express Framework.

Some of the key features of the application included a menu system for selecting items and customizing orders, a payment gateway for processing payments, and a real-time order tracking system for both customers and restaurant staff. The application was also integrated with a loyalty program system that rewarded customers for repeat orders and referrals.
